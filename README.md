# Discord Bot (sirBot)

## Installation

1. run `npm install`
2. run `node bot.js`

## Commands

Commands for this bot `!<command> [argument]`

| Command                        | Description                             |
| ------------------------------ | --------------------------------------- |
| `!reddit [subreddit]`          | replies with current top subreddit post |
| `!reddit-search [search term]` | replies with first search result        |
